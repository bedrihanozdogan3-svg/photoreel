## Proje Bilgisi
- **Proje:** PhotoReel AI — E-ticaret ürün fotoğraflarından Instagram/TikTok reels videosu üreten uygulama
- **Geliştirici:** Bedrihan Özdoğan (@bedrihan_ilyas_140525)
- **Stack:** HTML5 Canvas + Gemini API + Claude API + Jamendo müzik + Cloud Run
- **Repo:** github.com/bedrihanozdogan3-svg/photoreel
- **Cloud URL:** https://photoreel-194617495310.europe-west1.run.app

## Aktif Özellikler
- Otonom / Manuel video üretim modları
- 50+ geçiş efekti, beat-sync müzik, AI Director
- Trend DB entegrasyonu (9 kategori)
- Meta-Şablon sistemi (6 şablon)
- Öğrenme sistemi (13 veri türü, 100 beğeni sonrası optimize)
- Instagram API (trend hashtag analizi)

## Bilinen Sorunlar
1. Birden fazla fotoğraf yüklenince sadece biri videoda
2. Dinamik geçişler yeterince CapCut tarzı değil
3. Müzik bazen tekrar ediyor
4. Bitiş kartı bazen görünmüyor
